## ğŸ§  Why is Arduino under "Embedded Systems"?

Although Arduino provides high-level abstraction and beginner-friendly tools, it is fundamentally based on **microcontroller programming**, which is the core of embedded systems.

> Think of Arduino as a **gateway** to deeper embedded development.

By working with Arduino, I learned:
- How sensors and actuators work
- Basic digital/analog I/O
- Communication protocols like I2C, SPI, and UART
- Real-time behavior, delays, and interrupts
